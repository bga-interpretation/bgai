## Normwerte (Beispiel)
| Parameter | Normalbereich |
|---|---|
| pH | 7,35–7,45 |
| pCO₂ | 35–45 mmHg |
| HCO₃⁻ | 22–26 mmol/L |
| BE | -2 bis +2 mmol/L |

## Was misst was?
- pCO₂ → respiratorische Komponente
- HCO₃⁻/BE → metabolische Komponente

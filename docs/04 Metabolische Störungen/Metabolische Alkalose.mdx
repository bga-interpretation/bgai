**1** erwartete Kompensation berechnen: erwarteter PaCO<sub>2</sub> (mmHg) = (0,7 x HCO<sub>3</sub><sup>-</sup>) + 20
- gemssen +/- 5 berechnet → **adäquate Kompensation**
- gemessen >5 berechnet → **zusätzliche respiratorische Azidose**
- gemessen &lt;5 berechnet → **zusätzliche respiratorische Alkalose**

**2** Urin-Cl<sup>-</sup> &lt; 10-20 mmol/l → **Volumenmangelbedingt: Erbrechen/Magensonde, Diuretika (früh nach Absetzen), Volumenmangel (Kontraktionsalkalose)**

<span className="small-text">Mechanismus: (1) Verlust von H<sub>+</sub> und Cl<sup>-</sup> (2) RAAS-Aktivierung → HCO<sub>3</sub><sup>-</sup>-Retention</span>

- Urin-Cl > 20 mmol/l → **Volumenunabhängig: Mineralokortikoid-Exzess (Hyperaldosteronismus, Cushing), Hypokaliämie, persistierende Diuretikawirkung, seltene Ursachen (Bartter-Syndrom, Gitelman-Syndrom, Liddle-Syndrom)**

**3** Volumenstatus beurteilen
- Hypovolämie → **Erbrechen, Diuretika**
- Normo-/Hypervolämie + Hypertonie → **Aldosteron-Exzess**

**4** Kalium berücksichtigen

<span className="small-text">Hypokaliämie verstärkt die metabolische Alkalose durch (1) intrazelluläre H<sup>+</sup>-Verschiebung und (2) renale HCO<sub>3</sub><sup>+</sup>-Retention</span>
- Hypokaliämie + Hypertonie → **Hyperaldosteronismus**
- Hypokaliämie + normaler Blutdruck → **Bartter/Gitelman, Diuretika**

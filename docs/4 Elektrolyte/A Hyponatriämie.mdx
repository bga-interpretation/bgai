<div class="center">*Anamnese: 5 wichtige Fragen bei Hyponatriämie*</div>
**0** Zufuhr von zu viel Wasser? (Marathonläufer, Ecstacy/MDMA…) → **Polydipsie**
- Diuretika-Therapie → **Diuretika induzierte Hyponatriämie**
- Gastrointestinale Symptomatik (Diarrhoe, Erbrechen) mit Flüssigkeitsverlust? → **Diarrhoe** 
- Klinische Zeichen einer Herzinsuffizienz, einer Leber- oder Nierenerkrankung?	→ **?**
- Grunderkrankung, die zu einem SIADH führen kann? → **?**
- /	→ **2**  

<div class="center">*Vorüberlegungen, Muster*</div>

**2** Glukose ↓ + Na ↓	→ **an Nebenniereninsuffizienz denken!**  
<span className="small-text">Cortisol ↓ => ADH ↑ => Na ↓, Eine Addison-Krise sieht man heute eher nicht mehr. Meist eher bei Langzeitbehandlung mit Kortikosteroiden + Stress. (Am meisten beobachtet bei AIDS-Epidemie), oder: Polyglandulärer Tumor, Hypophysentumor, sekundäre Nebenniereninsuffizienz</span>  
– K ↑ + keine ARBs, keine ACE-Hemmer und keine Nieren "on fire" (dann sind schon 98% aller K ↑ ausgeschlossen) → **an primäre Nebenniereninsuffizienz denken!**  
<span className="small-text">Aldosteron ↓ verhindert Sekretion von K => K ↑</span>  
<span className="small-text">aber die Nebennieren sind ausgelöscht => Cortison ↓ => Hyponatriämie</span>  
<span className="small-text">(Achtung: nicht: Verlust von Aldosteron => Na ↓, sondern: Verlust von Cortisol => ADH ↑ => Na ↓)</span>  
<span className="small-text">bei der sekundären Nebenniereninsuffizienz wird Aldosteron gebildet!</span>  

– BUN, Kreatinin ↑ + keine Dialyse + Nieren "on fire" → **Niereninsuffizienz**  
<span className="small-text">Wasser wird getrunken und kann nicht ausgeschieden werden</span>  
– anders → **weiter mit 3**  

<div class="center">*Ausschluss einer hypertonen Hyponatriämie*</div>

**3** > 275 mOsm/kg	→ **nicht-hypotone Hyponatriämie, 4**  
– < 275 mOsm/kg	→ **hypotone („echte“) Hyponatriämie, 7**  

**4** Glukose ↑ → **Hyperglykämie, diabetische Ketoazidose**  
<span className="small-text">100 mg/dl ↑ … 2,4 mmol/l Na ↓</span>  
– Anamnese, Patientenakte → **Mannitol, Kontrastmittel, Glycin (TUR-Syndrom), Histidin-Tryptophan-Ketoglutarat (HTK) (Organtransplantation, Herzchirurgie), Maltose (z.B. in einigen IVIG-Präparaten)**  
[siehe oben?]  
– Osmolale Lücke > 10-15 → **Alkohol (Ethanol, Methanol), Ethylenglykol, Mannitol, Kontrastmittel**  
– Osmolale Lücke < 10-15 → **5**  

**5** Harnstoff ↑ → **Urämie**  
– Lipide ↑ (Triglyzeride, Cholesterin) → **Pseudohyponatriämie bei Hyperlipidämie**  
– Proteine ↑ (Gesamtprotein, Albumin) → **Pseudohyponatriämie bei Paraproteinämie, 6**  

**6** Monoklonale Gammopathie → **Multiples Myelom**  
– Intravenöse Immunoglobuline → **Immunglobulintherapie**  

<div class="center">*Ausschluss einer akuten Hyponatriämie*</div>

**7** < 48 h bestehend und/oder einer der folgenden Kriterien erfüllt → **akute Hyponatriämie**  
<span className="small-text">Postoperative Phase, nach Prostataresektion, nach endoskopischer Uterusresektion, Polydipsie, körperliche Belastung, kürzlich verordnete Thiazide, 3,4-Methylendioxymethamphetamin (MDMA, Ecstasy/XTC), Koloskopievorbereitung, Cyclophosphamid (i.v.), Oxytocin, kürzlich begonnene Desmopressin-Therapie, kürzlich begonnene Terlipressin-Therapie, Vasopressin</span>  
– > 48 h bestehend oder Zeitdauer unbekannt, keiner der Kriterien erfüllt → **chronische Hyponatriämie, 8**  

<div class="center">*Harnosmolarität und Harn-Natriumkonzentration*</div>  

**8** Harnosmolarität < 100 mOsm/kg, niedriges effektives arterielles Blutvolumen → **Polydipsie, geringe Zufuhr von Elektrolyten, Bierpotomanie**  
– Harnosmolarität > 100 mOsm/kg → **9**  

**9** Harn-Natriumkonzentration &lt;30, geringes effektives arterielles Blutvolumen → **10**  
– Harn-Natriumkonzentration >30 → **11**  

**10** Extrazelluläres Volumen ↑ → **Herzinsuffizienz, Leberzirrhose, nephrotisches Syndrom**  
– Extrazelluläres Volumen ↓ → **Diarrhoe und Erbrechen, Third spacing, „remote diuretics“**  

**11** Diuretika oder Nierenerkrankung → **Diuretika, Nierenerkrankung**  
– /	**12**  
**12** Extrazelluläres Volumen ↑ → **Erbrechen, primäre Nebenniereninsuffizienz, renales Salzverlustsyndrom, zerebrales Salzverlustsyndrom, okkulte Diuretika**  
– Extrazelluläres Volumen normal → **SIADH, sekundäre Nebenniereninsuffizienz, Schilddrüsenunterfunktion, okkulte Diuretika**
